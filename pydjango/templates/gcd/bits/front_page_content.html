New Features for December 2010!

  • Deletion support: All indexers may delete items from the database.
  • Wraparound and gatefold cover support: You can now upload and display wraparound and gatefold covers.
  • Advanced search for covers: Searches for covers now show the cover images.
  • Multiple issue changes support: You can make changes to the issue information across multiple issues at the same time through Advanced Search results.
  • ISBN support: We've added a field to track ISBNs.
  • Improved sequence insertion: It's now easier to insert or reorder sequences anywhere in an issue.
  • International statistics and a Daily Changes page: See what's changed daily.
  • Change history view: Detailed item change history since December 2009 is now visible. Pre-December 2009 change history (to the extent that it was recorded) will be imported in the near future.
  • Number of indexed issues per series: In all series lists we now display the number of indexed issues to easily find series which need work.
